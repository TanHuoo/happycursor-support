# 喜标 - macOS 光标定制工具
 安装地址，已上架Appstore：https://apps.apple.com/cn/app/%E5%96%9C%E6%A0%87/id6748666239?mt=12

 
> **版本 1.1.2** | 让您的 Mac 光标更有个性

一款简洁易用的 macOS 应用，提供自定义光标、聚光灯效果、氛围特效、幻灯片演示和屏幕标注等功能，让您的桌面体验更加个性化和有趣。

## ✨ 主要功能

### 🎯 自定义光标
- **多种样式**: 支持默认形状和自定义图片
- **个性配置**: 自由调整大小、颜色、圆角
- **特效增强**: 发光效果、点击动画、拖尾轨迹
- **格式支持**: PNG、SVG、GIF 图片格式
- **实时响应**: 毫秒级参数调整，丝滑流畅体验
- **多屏支持**: 完美支持多显示器环境

### 🔍 聚光灯功能
- **快捷激活**: 组合键快速启用聚光灯效果
- **演示利器**: 突出显示屏幕重点区域，适合教学演示
- **灵活调节**: 可调整聚光灯大小和遮罩透明度
- **幻灯片集成**: 在幻灯片播放时自动显示在最高层级

### 🎆 氛围效果
- **庆祝动画**: 双击修饰键触发彩带庆祝效果
- **场景适用**: 完美适配演讲、教学、庆祝场合
- **自定义设置**: 可调节动画持续时间和彩带数量

### 🎬 幻灯片演示
- **一键激活**: 点击状态栏图标或图片缩略图立即开始全屏演示
- **智能图片管理**: 支持拖拽上传、文件选择器，最多10张图片
- **全屏沉浸体验**: 深色背景突出图片内容，专业演示效果
- **GIF动画支持**: 全面支持GIF动图，在幻灯片中流畅播放动画效果

### ✏️ 屏幕标注
- **多种工具**: 画笔、矩形、圆形、箭头、文字标注
- **实时绘制**: 支持实时绘制和编辑
- **颜色选择**: 七种预设颜色，快速切换
- **线宽调节**: 自由调整绘制线宽
- **文字标签**: 支持多行文字输入，美观的气泡样式
- **撤销功能**: 右键快速撤销，支持多步撤销
- **保存分享**: 支持保存为图片和复制到剪贴板

## 🔧 系统要求

- **操作系统**: macOS 13.0 (Ventura) 或更高版本
- **权限需求**: 输入监控权限（全局快捷键）、屏幕录制权限（聚光灯功能）
- **架构支持**: Apple Silicon (M1/M2/M3) 和 Intel 64位

## 📖 详细使用教程

### 🎯 自定义光标设置

#### 启用自定义光标
1. 打开应用，进入"光标设置"
2. 启用"自定义光标"开关
3. 选择样式并调整参数

#### 光标样式选择
- **默认形状**: 选择圆形、方形等基础形状
- **自定义图片**: 上传PNG、SVG、GIF格式的图片
- **GIF动画**: 支持动态光标效果

#### 参数调整
- **大小**: 调整光标显示大小（10-100像素）
- **颜色**: 选择光标填充颜色
- **圆角**: 调整光标圆角程度
- **发光效果**: 开启/关闭发光效果
- **点击动画**: 开启/关闭点击缩放动画
- **拖尾效果**: 开启/关闭光标拖尾轨迹

### 🔍 聚光灯功能使用

#### 启用聚光灯
1. 进入"辅助功能" → "聚光灯"
2. 启用功能并设置快捷键（默认：⌃⌥M）
3. 按住快捷键激活聚光灯效果

#### 聚光灯操作
- **激活**: 按住设置的快捷键
- **移动**: 移动鼠标调整聚光灯位置
- **调整大小**: 使用鼠标滚轮调整聚光灯大小
- **调整透明度**: 在设置中调整遮罩透明度

#### 适用场景
- **教学演示**: 突出显示重点内容
- **屏幕录制**: 引导观众注意力
- **远程协助**: 指示操作位置

### 🎆 氛围效果使用

#### 启用氛围效果
1. 进入"辅助功能" → "氛围"
2. 启用功能并选择触发键（Control、Option、Command、Shift）
3. 双击选定的修饰键触发效果

#### 效果设置
- **动画时长**: 调整彩带动画持续时间（1-5秒）
- **彩带数量**: 调整彩带密度（50-200个）
- **触发间隔**: 双击间隔需在0.5秒内完成

#### 适用场景
- **演讲庆祝**: 重要时刻的庆祝效果
- **教学互动**: 学生回答正确时的鼓励
- **直播效果**: 增加直播间的互动氛围

### 🎬 幻灯片演示使用

#### 添加图片
1. 进入"幻灯片设置"
2. 点击"添加图片"或直接拖拽图片到应用
3. 支持PNG、JPEG、GIF格式，最多10张图片

#### 开始演示
- **方式一**: 点击状态栏的幻灯片图标
- **方式二**: 点击图片缩略图
- **方式三**: 使用全局快捷键（需先设置）

#### 演示操作
- **切换图片**: 使用左右方向键
- **退出演示**: 按ESC键
- **全屏显示**: 自动进入全屏模式，深色背景

#### 图片管理
- **重新排序**: 拖拽调整图片顺序
- **删除图片**: 选中图片后点击删除按钮
- **清空所有**: 一键清空所有图片

### ✏️ 屏幕标注使用

#### 启动标注
1. 进入"辅助功能" → "标注"
2. 启用功能并设置快捷键（默认：双击Option键）
3. 使用快捷键启动标注模式

#### 工具使用

##### 🖌️ 画笔工具
- **选择工具**: 点击画笔图标
- **绘制**: 按下鼠标左键进行自由绘制
- **撤销**: 右键点击撤销上一步操作
- **线宽调整**: 使用滑块调整画笔粗细

##### 📐 矩形工具
- **选择工具**: 点击矩形图标
- **绘制**: 拖拽鼠标绘制矩形
- **正方形**: 按住Shift键绘制正方形
- **颜色选择**: 点击颜色按钮选择绘制颜色

##### ⭕ 圆形工具
- **选择工具**: 点击圆形图标
- **绘制**: 拖拽鼠标绘制圆形
- **正圆**: 按住Shift键绘制正圆
- **填充**: 可选择是否填充圆形

##### ➡️ 箭头工具
- **选择工具**: 点击箭头图标
- **绘制**: 拖拽鼠标绘制箭头
- **方向**: 箭头方向跟随拖拽方向
- **大小**: 箭头大小根据拖拽距离自动调整

##### 📝 文字工具
- **选择工具**: 点击文字图标
- **创建标签**: 点击空白区域创建文字输入框
- **输入文字**: 在输入框中输入文字内容
- **确认文字**: 按回车键确认，按ESC键取消
- **多行文字**: 按回车键换行
- **多个标签**: 可以创建多个文字标签

#### 颜色选择
- **预设颜色**: 七种预设颜色（红、蓝、黄、绿、青、橙、紫）
- **快速切换**: 点击颜色按钮立即切换
- **选中状态**: 当前颜色按钮会高亮显示

#### 操作技巧
- **撤销操作**: 右键点击或点击撤销按钮
- **清空所有**: 点击清空按钮清除所有标注
- **保存标注**: 点击保存按钮保存为图片
- **复制到剪贴板**: 自动复制标注内容到剪贴板

#### 退出标注
- **ESC键**: 按ESC键退出标注模式
- **工具栏**: 点击工具栏的退出按钮

### ⌨️ 快捷键设置

#### 全局快捷键
- **聚光灯**: 默认 ⌃⌥M（Control+Option+M）
- **氛围效果**: 双击修饰键（Control/Option/Command/Shift）
- **标注功能**: 双击修饰键（Control/Option/Command/Shift）
- **幻灯片**: 双击修饰键（Control/Option/Command/Shift）

#### 快捷键冲突处理
- 系统会自动检测快捷键冲突
- 冲突时会显示提示对话框
- 建议为不同功能设置不同的修饰键

#### 修改快捷键
1. 进入对应功能的设置页面
2. 点击修饰键按钮选择新的快捷键
3. 系统会自动检测冲突并提示

## 📊 技术架构亮点

### 🏗️ 模块化设计
项目采用模块化架构，将复杂功能拆分为专门的管理器：
- **CursorManager**: 核心协调器，统一管理所有子模块
- **CursorRenderer**: 专门负责光标的绘制和视觉效果
- **ImageManager**: 专门负责图像加载、处理和缓存
- **SystemCursorController**: 专门负责系统光标的控制
- **CursorWindowManager**: 专门负责窗口和视图管理
- **CursorConfiguration**: 专门负责配置状态的统一管理
- **CursorStateMachine**: 专门负责状态转换和生命周期

### ⚡ 性能优化

- **内存优化**: 专门的内存管理器优化资源使用
- **多屏支持**: 优化的多显示器支持，确保性能稳定
